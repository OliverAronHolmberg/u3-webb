HTML och CSS element

HTML

Meta

  • HTML : Huvudelement som innehåller htmlfilen
  • Head : En contianer för metadata, data om data
  • Meta : Finns för att ge data om stuktur
  • Title : Titeln/namnet på hemsidan
  • Style : Innehåller css-stilen på hemsidan

Semantiska Element

  • Title : Namnger titeln av dokumentet/sidan
  • Main : innehållet på sidan, innehåller flera sections
  • Nav : En del av sidan vilkets mening är att ge navigations länkar
  • Section : En del av dokumentet

Text och Rubriker

  • h1-6 : Headers i olika storlekar, rubriker med andra ord
  • p : Paragraph element/brödtext

Layout

  • Header : Sidhuvud ineehåller information om hemsidan
  • Body : Documentets body
  • Footer : Innehållet på slutet av sidan, som länkar

CSS

Typsnitt och stilar

  • font-family : Använd detta till att ända textens font
  • Font-size : Mna kan använda detta till att ändra storleken på text
  • font-weight : Använd detta för att göra texter BOLD

Färger

  • color : Använd detta för att ändra färg på ett element, som text
  • background-color : Genom detta kan man ändra bakgrundsfärgen på elementet
  • background : Kan använda detta för att ändra egenskaper i bakgrunden, som färg

Layout

  • margin-left/right/up/down : Använd detta för att ändra specifika mellanrum på olika ställen
  • margin : Använd detta till att kontrollera mellanrummet mellan elementet och skärmen
  • padding : Använd detta för att skapa mellanrum mellan elementväggarna och children i elementet
  • border : Använd detta för att skapa en border
  • border-radius : Använd för att kontrollera hur rund border är
  • align-items : Använd för att berätta hur elementen i ett annat element ska ligga
  • justify-content : Använd detta för att tex. lägga space mellan alla element i elementet
  • gap : Använd detta för att styra hur mycket space mellan element
  • list-style : använd detta för att syla listor
